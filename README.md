## Introducción

### Contexto
El proyecto **Bendito Café** consiste en una plataforma web diseñada para promover y presentar los servicios de una cafetería. Ha sido estructurada para ofrecer una navegación intuitiva y una visualización atractiva de los productos e instalaciones.

### Objetivo
Desarrollar una interfaz moderna, dinámica y responsiva que permita a los usuarios explorar el menú, conocer la historia de la cafetería, interactuar con la galería de imágenes y acceder fácilmente a los medios de contacto.

### Alcance
El sistema comprende la interfaz de usuario (Frontend), construida en base a páginas estáticas enlazadas, que operan de manera independiente sin requerir un procesamiento de backend activo para su visualización base.

---

## Arquitectura del Proyecto

### Tecnologías Empleadas

* **HTML5:** Estructuración semántica del contenido de cada página.
* **CSS3:** Estilización, diseño responsivo y presentación visual.
* **JavaScript (Vanilla):** Interactividad del lado del cliente.

### Estructura Documental

El repositorio se organiza de la siguiente manera:

* `index.html`: Página principal y portada de la aplicación (Landing Page).
* `about.html`: Página informativa sobre la visión y la historia del establecimiento.
* `menu.html`: Catálogo detallado de bebidas y alimentos.
* `gallery.html`: Galería visual de los productos y el local.
* `contact.html`: Información de ubicación y formulario de contacto.
* `style.css`: Hoja de estilos centralizada.
* `script.js`: Archivo principal de control de eventos y manipulación del DOM.

## Instalación y Uso

### Requisitos Técnicos
* Navegador web moderno (Google Chrome, Mozilla Firefox, Safari, Microsoft Edge).

### Procedimiento de Ejecución
1. Obtener los archivos fuente del repositorio.
2. Abrir el archivo `index.html` directamente en el navegador web preferido.
3. Para desarrollo, se recomienda utilizar un servidor local (por ejemplo, Live Server en Visual Studio Code) para evitar restricciones de políticas CORS en caso de usar módulos adicionales en el futuro.
